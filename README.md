# moveboxs2.0
## 推箱子EGE版本
### 1.特点  
>(1).丑  
(2).实现了多步撤销  
`实现原理每走一步记录箱子和人的坐标，箱子用二维数组，人用一维数组。`  
(3).可以吃吐箱子  
`由于关卡难度太大，降低难度。`  
(4).关卡锁  
`玩家必须通关前一关才能玩下一关。`  
(5).地图编辑器  
`可以自定义地图。`  
(6).可以选择人物  
`多种风格不同的人物`
(7).音效
`不同的按键有不同的音效，还可以可以控制开关`
